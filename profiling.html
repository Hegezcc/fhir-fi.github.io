Finnish FHIR Base Profiles

The profiling of FHIR resources and use cases for Finland has been started!

The first phase aims to polish and publish profiles from existing use cases and implementations. It is expected to complete by the end of March, 2023.

GitHub Repositories

There are two GitHub repos where artifacts are being created. One for Finnish Base Profiles, one for Finnish Implementation Guide for SMART App Launch.

Current Snapshots

The current state of the implementation guide including the base resources is published at https://fhir.fi/finnish-base-profiles/.

The current state of the implementation guide for SMART App Launch is published at https://fhir.fi/finnish-smart/.

The final results of both work streams will most likely eventually be published somewhere else, for instance in Simplifier.
