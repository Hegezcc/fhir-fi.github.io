Finnish FHIR Base Profiles

The profiling of FHIR resources and use cases for Finland has been started!

There are two implementation guides being prepared.

  • The Finnish Base Profiles IG specifies the Finnish FHIR base profiles. See the source code and the latest snapshot.
  • The Finnish Implementation Guide for SMART App Launch describes how the SMART specification is applied in Finland. See the source code and the latest snapshot.

Both implementation guides have been through the ballot of HL7 Finland membership and have passed the vote. They are being polished for publication and are expected to be formally published in June, 2023.
